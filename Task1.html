Javascript Datatypes

JavaScript provides different data types to hold different types of values. There are two types of data types in JavaScript.
1. Primitive data type
2. Non-primitive (reference) data type

Primitive data type

There are five types of primitive data types in JavaScript. They are as follows:
  • Number -Represents numeric values e.g. 100
  • BigInt - Represents numeric values e.g. 100256236e7833728n
  • String - Represents sequence of characters e.g. "hello"
  • Boolean -Represents boolean value either false or true

    • Non-primitive (reference) data type

  • Null - Represents null i.e. no value at all(absence of an value)
  • Undefined - Represents undefined value( uninitialized variable)
  • Symbol - it is a unique and immutable primitive value that can be used as the key of an object property
  • Object - The collection of properties where each property is a key value pair.

    • Operators

    JavaScript operators are symbols that are used to perform operations on operands.
    There are following types of operators in JavaScript.

    Arithmetic Operators

    Arithmetic operators are used to perform arithmetic operations on the operands such as addition(+), Subtraction(-),Multipliocation(*), Division(/), Modulus(%) Exponential(**), Increment(++), Decrement(--) .
